Instalar bibliotecas de terceros

Hay tres maneras de hacer uso de bibliotecas y/o frameworks de terceros.

Bootstrap

Vía CDN, basta con copiar los enlaces de los recusros deseados, en este caso archivos CSS y JavaScript, y posteriormente pegarlos en el archivo index.html.

De manera local, es necesario descargar la biblioteca y/o framework, posteriormente pegar los recursos en la carptea assets de nuestro proyecto, quedando de la siguiente manera.

Finalmente, ahora los links y scripts deben apuntar a nuestra ruta local:

<link rel="stylesheet" href="assets/bower_components/bootstrap/dist/css/bootstrap.min.css"> 
<!-- Font Awesome -->
<link rel="stylesheet" href="assets/bower_components/font-awesome/css/font-awesome.min.css">
<!-- Ionicons -->
<link rel="stylesheet" href="assets/bower_components/Ionicons/css/ionicons.min.css">
<!--===============================================================================================-->
<link rel="stylesheet" href="assets/fonts/iconic/css/material-design-iconic-font.min.css">
<!-- Theme style -->
<link rel="stylesheet" href="assets/css/AdminLTE.css">
<!-- AdminLTE Skins. We have chosen the skin-blue for this starter
        page. However, you can choose any other skin. Make sure you
        apply the skin class to the body tag so the changes take effect. -->
<link rel="stylesheet" href="assets/css/skins/skin-black-light.min.css">
<!-- REQUIRED JS SCRIPTS -->
<!-- jQuery 3 -->
<script src="assets/bower_components/jquery/dist/jquery.min.js"></script>
<!-- Bootstrap 3.3.7 -->
<script src="assets/bower_components/bootstrap/dist/js/bootstrap.min.js"></script>
<!-- AdminLTE App -->
<script src="assets/js/adminlte.min.js"></script>

Vía NPM, instalamos la biblioteca y/o framework mediante la consola de comandos

$npm install bootstrap --save
$npm install jquery --save

Posteriormente, dentro del archivo angular.json, agremamos las referencias a los recursos CSS y JavaScript, los cuales se encuentran en la carpeta node_modules. En el apartado de styles, agregamos los recursos CSS y en scripts, agregamos los recursos JavaScript que correspondan.

"styles": [
    "src/styles.css",
    "node_modules/bootstrap/dist/css/bootstrap.min.css"
],
"scripts": [
    "node_modules/jquery/dist/jquery.slim.min.js",
    "node_modules/popper.js/dist/umd/popper.min.js",
    "node_modules/bootstrap/dist/js/bootstrap.min.js"
]
npm install bootstrap --save
npm install bootstrap --save
npm install bootstrap --save
