Directivas *ngIf y *ngFor

Las directivas *ngIf y *ngFor son atributos que podemos agregarle a los elementos HTML que nos permiten en el caso del *ngIf condicionar si dicha marca debe agregarse a la página HTML.

La directiva *ngFor nos permite generar muchos elementos HTML repetidos a partir del recorrido de un arreglo de datos.

*ngFor

Una directiva estructural que representa una plantilla para cada elemento de una colección. La directiva se coloca en un elemento, que se convierte en el padre de las plantillas clonadas.
*ngIf

Una directiva estructural que incluye condicionalmente una plantilla basada en el valor de una expresión obligada a Boolean. Cuando la expresión se evalúa como verdadera, Angular representa la plantilla proporcionada en una cláusula then, y cuando es falsa o nula, Angular representa la plantilla proporcionada en una cláusula else opcional. La plantilla predeterminada para la cláusula else está en blanco.

<div *ngIf="heroe.casa === 'Marvel';else DC">
    <img src="./assets/img/marvel.png" class="img-fluid" alt="Marvel">
</div>
<div *ngIf="condition; else elseBlock">Content to render when condition is true.</div>
<ng-template #elseBlock>Content to render when condition is false.</ng-template>
