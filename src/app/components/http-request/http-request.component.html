Peticiones Http
  • En el app.module.ts importamos HttpClientModule
  • Agregamos el modulo a imports
  • Podemos trabajar y/o ocupar HttpClient desde el componente que deseamos ocupar, sin embargo siempre será mejor trabajar desde un servicio. Importamos HttpClient, HttpHeaders y map (es muy común trabajar con estos modulos)
  • Si se van a relizar muchas peticiones a una misma url, estas se pueden centralizar. En caso de requerir alguna autorización (un token por ejemplo), aquí mismo se puede realizar.
  • El resto es codificar nuestras peticiones http según nos convenga. Es una buena practica filtrar los resultados con el operador map.
Suscribirse a peticiones http
  • Inyectamos el servicio en el constructor del componente desde donde vamos a realizar las peticiones http.
  • Nos subscribimos a la petición y enseguida podemos realizar un manejo de error del lado del cliente.
