@Input

Decorador que marca un campo de clase como una propiedad de entrada y proporciona metadatos de configuración. La propiedad de entrada está vinculada a una propiedad DOM en la plantilla. Durante la detección de cambios, Angular actualiza automáticamente la propiedad de datos con el valor de la propiedad DOM.

  • Importar Input de @angular/core
  • La declaración es similar a cualquier otra variable.

Notas de uso: Puede proporcionar un nombre opcional para usar en las plantillas cuando se instancia el componente, que se asigna al nombre de la propiedad enlazada. De forma predeterminada, el nombre original de la propiedad enlazada se usa para el enlace de entrada.
